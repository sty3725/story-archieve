# 프로젝트 개요

## 프로젝트명

Story Archive

## 제품 정의

작가와 글쓰기 학습자가 원고를 작성·정리하고, 작업 이력과 최근 문서를 관리하며, 강의실 기능을 함께 사용할 수 있는 웹 애플리케이션.

## 핵심 목표

1. 사용자가 계정을 생성하고 로그인할 수 있다.
2. 로그인한 사용자만 대시보드와 개인 데이터에 접근한다.
3. 원고를 생성·조회·수정·삭제할 수 있다.
4. 원고 변경 이력을 버전으로 보관한다.
5. 에디터에서 자동 저장과 불러오기가 가능하다.
6. 최근 문서와 주요 통계를 대시보드에서 확인한다.
7. 강의실에서 학습 콘텐츠 또는 과제를 관리한다.

## 메인 내비게이션

- 대시보드
- 아카이브
- 강의실

## 보조 내비게이션 후보

- 최근 문서
- 즐겨찾기 또는 고정 문서
- 휴지통

## 주요 사용자 흐름

### 신규 사용자

1. 회원가입
2. 가입 성공 확인
3. 로그인
4. 인증 정보 발급 및 저장
5. 대시보드 이동
6. 첫 원고 생성

### 기존 사용자

1. 로그인
2. 대시보드 진입
3. 최근 문서 확인
4. 원고 편집
5. 자동 저장
6. 버전 이력 확인
