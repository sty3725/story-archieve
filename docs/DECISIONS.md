# 설계 결정 기록

## 확정된 결정

### 기술 스택

- 백엔드: FastAPI
- 데이터베이스: PostgreSQL
- 프론트엔드: HTML, CSS, Vanilla JavaScript

### 주요 메뉴

- 대시보드
- 아카이브
- 강의실

### 숏컷 방향

- 최근 문서
- 즐겨찾기 또는 고정 문서
- 휴지통

### 인증 방향

- 회원가입 다음 단계는 로그인과 대시보드 연결
- 대시보드는 보호 페이지
- 백엔드 API와 프론트엔드 페이지를 모두 보호
- 비밀번호 평문 저장 금지

### UI 방향

- 왼쪽 메인 사이드바
- 중앙 작업 영역
- 오른쪽 보조 사이드바
- 오른쪽 보조 사이드바는 화면 상단부터 독립 영역으로 구성
- 상단 저장 상태 표시줄은 중앙 작업 영역까지만 차지
- 자유 색상 선택보다 6개 고정 테마 우선
- 블랙 계열과 아이보리 화이트 계열 테마 포함

## 아직 미확정

- JWT와 쿠키 중 인증 방식
- ORM 및 마이그레이션 도구
- 자동 저장 주기
- 원고 삭제 방식
- 강의실의 역할과 권한 모델
- 버전 생성 조건
- 배포 환경
